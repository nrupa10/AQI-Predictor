# 🌿 AQI Predictor

An AI-powered Air Quality Index prediction web app built with Streamlit and scikit-learn.

## Features
- Predicts AQI from 12 pollutant inputs
- Random Forest model trained on India's city-day dataset
- Health recommendations based on AQI category
- Interactive speedometer gauge

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack
- Python, Streamlit, scikit-learn, Plotly, Pandas
- Dataset: Kaggle India City Day AQI Dataset
